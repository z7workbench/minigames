# Audio Assets Directory

Place your sound files here:

## Required Sounds
- `button_tap.mp3` - Button click sound
- `dice_roll.mp3` - Dice rolling sound
- `correct.mp3` - Correct answer sound
- `wrong.mp3` - Wrong answer sound
- `win.mp3` - Win game sound
- `lose.mp3` - Lose game sound

## Format
- MP3 or WAV format recommended
- Sample rate: 44.1kHz or 48kHz
- Bit depth: 16-bit
- Mono or stereo acceptable
